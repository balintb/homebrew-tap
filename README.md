# homebrew-tap
Homebrew tap for @balintb

To add the tap, run

```console
brew tap balintb/tap
```

## Formulae

### [`clipsyboogie`](https://github.com/balintb/clipsyboogie)

```console
brew install balintb/tap/clipsyboogie
```
